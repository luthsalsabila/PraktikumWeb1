# Praktikum_Web1

## HTML

HTML adalah singkatan dari HyperText Markup Language, adalah bahasa markup untuk membuat dan mengatur konten web. Dengan tag-tag khusus, pengembang dapat menandai teks, gambar, dan elemen lainnya untuk disajikan secara terstruktur di halaman web, memungkinkan tampilan yang konsisten di berbagai platform dan perangkat.
